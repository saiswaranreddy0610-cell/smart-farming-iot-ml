# smart-farming-iot-ml
A Step Toward Smarter Farming! During my internship at Gitam School of Science (May–July 2025), I worked on “Crop Prediction Using Sensors and Cloud Integration.” Using Arduino Uno, I built a sensor-based IoT system to collect soil and weather data, sent it to a Flask cloud server, and used ML to suggest ideal crops.
